Privacy

If you have the Navbar messages feature enabled, Courses+ will connect to a server so it can retrieve the newest message. When it does this, information about what browser version you have, what version of Courses+ you have, and a random, unique ID is transmitted to the server. No personally-identifying information is sent. To disable this tracking, disable the Navbar messages component.
